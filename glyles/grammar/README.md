# Implemented Grammar in GlyLES

## Implemented Monomers

All following monomers are implemented in alpha, beta and undefined conformation of the C1 atom.

* Fuc
* Gal
* Gal3S
* Gal6S
* GalNAc
* GalNAc4S
* GalNAc6S
* Glc
* GlcA
* GlcNAc
* GlcNAc6S
* Man
* Neu5Ac (currently not working)
* Neu5Gc (currently not working)
* Tal
